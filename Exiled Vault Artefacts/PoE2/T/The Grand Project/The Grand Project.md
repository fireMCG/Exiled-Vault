# The Grand Project

![[Exiled Vault Artefacts/_Images/PoE2 - The Grand Project.png]]

## Description
"I see a pattern in these towers. Whoever the Precursors were, they sought to master this world, to shape it as they saw fit. My people had the same ambitions..."

## Details
- Adds Irradiated to a Map
- 1 use remaining
- Can only be applied to Precursor Tower Maps
- Completing the Tower makes all nearby Maps accessible
- Can be used in a personal Map Device to add modifiers to a Map.
