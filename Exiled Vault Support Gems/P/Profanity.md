## Profanity
Support
Chaos
Tier: 1
Level: 1
Support Requirements : +5 Int
Supports Skills you use yourself. Supported Skills consume a Corpse on use, dealing more Chaos Damage if they do. Cannot support Triggered Skills or Reservation Skills.
Supported Skills consume a Corpse within 5 metres on use, if possible
Supported Skills deal 30 % more Chaos Damage if consuming a Corpse
